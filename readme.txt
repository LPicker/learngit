这是我学习 git 的第一步

DOS命令：
mkdir learngit			创建文件目录 learngit

初始化一个Git仓库，使用git init命令。

添加文件到Git仓库，分两步：

第一步，使用命令git add <file>，注意，可反复多次使用，添加多个文件；

第二步，使用命令git commit，完成。


mkdir learngit 创建文件夹
cd learngit 进入learngit子目录
pwd一次 用于显示当前的目录
git init 初始化目录变成git可管理的仓库
git add readme.txt 如果建立不成功手动处理建立。
vi readme.txt 输入自己要输入的内容，保存推出
git add . 添加内容到文件当中 回车
git commit一次 弹出文件保存的主目录下 消除注释 保存退出
查看read.txt有自己添加的内容。 这是我自己的方法，在没有添加内容成功的情况下。请廖老师指导，为也是初学者。
